# Práctica de L-Systems

## Desarrollador
- Betanzos Reyes Gustavo Noel

## Descripción
En esta práctica se implementa un sistema L (Lindenmayer System) que genera patrones fractales mediante reglas de reescritura y una tortuga gráfica para su representación. El sistema L utiliza un axioma inicial y una serie de reglas para derivar cadenas de símbolos que son interpretadas como comandos gráficos para crear el patrón.

## Funcionalidad
El programa permite definir un axioma, un conjunto de reglas de reescritura, el número de iteraciones, y el ángulo de rotación. Utilizando estos parámetros, el programa deriva la cadena final y utiliza una tortuga gráfica para dibujar el patrón correspondiente en la pantalla.

## Uso
1. Ejecutar el script `LSystems.py`.
2. Ingresar el axioma en el campo correspondiente.
3. Ingresar las reglas de reescritura en el formato `X->Y`, una por línea.
4. Ingresar el número de iteraciones.
5. Ingresar el ángulo de rotación.
6. Presionar el botón "Iniciar Dibujo" para generar y visualizar el patrón.
