# Práctica de Redes Booleanas

## Desarrollador
- Betanzos Reyes Gustavo Noel

## Descripción
Esta práctica implementa una simulación de redes booleanas utilizando una interfaz gráfica desarrollada con Tkinter. La simulación se basa en la evaluación de una tabla de verdad para determinar los nuevos valores de los nodos en cada iteración.

## Funcionalidad
El programa permite al usuario ingresar el número de iteraciones y el número de nodos. Durante cada iteración, se generan valores aleatorios para los nodos y sus conexiones. Los valores de los nodos se actualizan en función de una tabla de verdad predefinida. Los resultados se muestran en una ventana con un widget de texto.

## Uso
1. Ejecutar el script `redes_booleanas.py`.
2. Ingresar el número de iteraciones en el campo "Número de iteraciones".
3. Ingresar el número de nodos en el campo "Número de nodos".
4. Presionar el botón "Calcular" para iniciar la simulación.
5. Los resultados de cada iteración se mostrarán en el widget de texto.
