# Práctica de Autómatas 1D

## Desarrollador
- Betanzos Reyes Gustavo Noel

## Descripción
En esta práctica se implementa la simulación de autómatas celulares unidimensionales utilizando las reglas de Wolfram. El objetivo es observar y analizar los patrones generados a partir de diferentes reglas y configuraciones iniciales.

## Funcionalidad
El programa permite al usuario ingresar una regla de Wolfram (un número entre 0 y 255) y un número de generaciones. A partir de estos parámetros, se genera una evolución del autómata celular que se visualiza en una gráfica. La evolución se calcula aplicando la regla seleccionada a cada celda de la generación anterior para determinar el estado de las celdas en la siguiente generación.

## Uso
1. Ejecutar el script `Automatas1D.py`.
2. Ingresar la regla de Wolfram (un número entre 0 y 255) en el campo correspondiente.
3. Ingresar el número de generaciones que se desean simular.
4. Presionar el botón "Generar" para visualizar la evolución del autómata celular.
